# Master's Thesis – LoRa vs ESP-NOW

This repository contains materials related to a master's thesis focused on a comparative analysis of LoRa and ESP-NOW wireless communication technologies.

The work includes both theoretical analysis and experimental evaluation based on a custom-designed research module built on the ESP32-C6 platform. The system enables measurements of transmission range, energy consumption, and communication quality in various environmental conditions.

The repository is structured into hardware and firmware components, along with the final thesis document describing methodology, implementation, and results.
